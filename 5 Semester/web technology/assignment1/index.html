Compatibility report

Print:
The most accurate showing of the print media query has Google Chrome. However, the other browser we tested (Edge, Internet Explorer 11 and Mozilla Firefox) are having problems with the css attributes 'orphans' and 'widows'.

Screen:
The normal rendering for sizes above the threshold of 320x480 (portrait and landscape) were accurate on all browers we tested. When testing if the browers can render the correct css for the mobile/tablet view, Edge and Internet Explorer both could not do that in any way. The Mozilla Firefox could render the css but struggeled with the tables and the navigation (nav tag). The only brower to properly work on all html elements was, again, Google Chrome.
